# NimbusAI ☁️ — AI-Powered Cloud Cost Optimizer

> Stop guessing. Start saving.

NimbusAI is a web app that analyzes cloud infrastructure 
billing data and identifies wasteful spending — delivering 
actionable cost-reduction recommendations with projected 
monthly savings on an interactive dashboard.

Inspired by what **Zarthi** does at enterprise scale, 
built for accessibility and simplicity.

---

## 🚀 Features

- 📂 **CSV Upload** — Upload real AWS Cost Explorer exports
- 🧪 **Sample Data** — Demo instantly with realistic mock data
- 🤖 **AI-Powered Analysis** — Detects idle and overprovisioned resources
- 📊 **Interactive Dashboard** — Spend breakdown, waste heatmap, savings meter
- 💡 **Recommendation Cards** — Specific actions with dollar savings per service
- ✨ **Smooth Animations** — Framer Motion powered transitions and micro-interactions

## 🛠️ Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/diiyeah/Nimbus-AI
cd Nimbus-AI
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run directly — no API key needed
App uses built-in mock AI recommendations out of the box.
No setup required.

```bash
npm run dev
```
